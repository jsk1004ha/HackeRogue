# 학켓몬 (Hackemon)

학켓몬은 Vite와 Vanilla JavaScript로 제작된 학술 테마의 포켓몬 스타일 로그라이크 게임입니다. 다양한 캐릭터를 수집하고 강화하며 끊임없이 밀려오는 웨이브를 돌파하세요!

## 🚀 주요 특징 (Key Features)

- **로그라이크 방식의 진행**: 매번 새로운 웨이브마다 랜덤하게 등장하는 적 트레이너와 야생 학켓몬을 상대합니다.
- **다양한 학켓몬**: 고유한 타입(물리, 화학, 공학, 생물, 지구, 수학, 정보, 일반), 특성(Abilities), 그리고 기술(Moves)을 가진 캐릭터들이 존재합니다.
- **전략적인 전투 시스템**: 타입 상성 관계를 활용하여 효율적인 전투를 수행하세요.
- **보상 및 상점 시스템**: 웨이브 클리어 시 무료 보상을 선택하거나, 상점에서 필요한 아이템을 구매하여 파티를 강화할 수 있습니다.
- **저장/불러오기 기능**: 로컬 스토리지를 활용하여 게임 진행 상황을 저장하고 나중에 다시 이어서 즐길 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

- **Language**: JavaScript (Modules)
- **Styling**: Vanilla CSS
- **Build Tool**: [Vite](https://vite.dev/)
- **Structure**: Single Page Application (SPA)

## 🏃 실행 방법 (How to Run)

프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

1. **의존성 설치**:
   ```bash
   npm install
   ```

2. **개발 서버 실행**:
   ```bash
   npm run dev
   ```

3. **빌드 (프로덕션)**:
   ```bash
   npm run build
   ```

## 🎮 게임 메커니즘 (Game Mechanics)

### 타입 상성 (Type Chart)
- **물리 (PHYSICS)**: 화학, 지구에 강함 / 공학, 수학, 정보에 약함
- **화학 (CHEMISTRY)**: 생물, 지구에 강함 / 물리에 약함
- **공학 (ENGINEERING)**: 물리, 정보에 강함 / 수학, 생물에 약함
- **생물 (BIOLOGY)**: 공학에 강함 / 화학, 지구에 약함
- **지구 (EARTH)**: 생물, 수학에 강함 / 물리, 화학에 약함
- **수학 (MATH)**: 물리, 공학에 강함 / 정보에 약함
- **정보 (INFO)**: 수학, 물리에 강함 / 공학에 약함
- **일반 (NORMAL)**: 상성 관계 없음

### 성장 시스템
- 전투를 통해 경험치(XP)를 획득하고 레벨업을 할 수 있습니다.
- 레벨업 시 능력치가 무작위로 상승하며, 특정 레벨마다 새로운 기술을 배울 수 있습니다.
